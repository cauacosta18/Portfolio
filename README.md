# Portfólio - Cauã Luiz Costa

## Descrição do Projeto
O projeto desenvolvido trata-se de uma landing page que atua como um portfólio, com o objetivo de apresentar meu perfil profissional, bem como minhas habilidades e outros projetos já desenvolvidos.

## Objetivo
O objetivo do projeto além de criar um portfólio de apresentação, foi também a prática e consolidação das minhas habilidades nas linguagens de HTML, CSS e JavaScript, bem como no design e organização do código.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## Estrutura do Projeto
- img: contém todas as imagens de todos os tipos do projeto.
    - certificados: contém as imagens dos certificados.
    - projetos: contém as screenshots dos projetos.
    - tecnologias: contém os logotipos das tecnologias.
- scripts: contém todos os arquivos .js do projeto.
- style: contém todos os arquivos .css do projeto.
    - base: contém os arquivos .css referentes às variáveis, reset de estilos padrão do navegador e a estilização da base do projeto.
    - components: contém arquivos referentes a pequenas porções de estilo do projeto.
    - sections: contém os arquivos de estilo das principais seções do projeto.

## Funcionalidades
- Menu interativo
- Efeitos visuais
- Slider apresentando projetos
- Responsividade

## Como Executar
Após extrair a pasta, é possível abrir a página de duas formas:

- 1ª forma:
    - Abra o arquivo index.html em um editor de texto e troque o trecho: (
    <script type="module" src="scripts/script.js"></script>
    <!-- <script src="scripts/fullScript.js"></script> -->
    ) por: (
    <!-- <script type="module" src="scripts/script.js"></script> -->
    <script src="scripts/fullScript.js"></script>
    ). Após isso, basta abrir o arquivo index.html no navegador.

- 2ª forma:
    - Utilize um servidor local para executar o projeto.
    - Sugestões
        - Instale a ferramenta XAMPP, copie e cole a pasta no caminho (xampp > htdocs > *cole aqui*), abra o painel de controle da ferramenta, clique no botão "start" ao lado do texto "Apache", abra o seu navegador e abra o link "http://localhost/portfolio/".
        - Instale a ferramenta VS Code, instale a extensão Live Server de Ritwick Dey, abra a pasta do projeto no VS Code, clique com o botão direito do mouse e selecione a opção Open with Live Server.

## Sobre o Desenvolvimento
O primeiro passo para o desenvolvimento do projeto foi a criação da estrutura das informações, logo após foi feito um wireframe para planejar o layout básico, um protótipo onde foi feita a escolha das fontes e paleta de cores, para então começar o desenvolvimento do código.

Com a base criada, o próximo objetivo foi desenvolver o HTML, então o CSS e por fim o JavaScript, depois, fazer as alterações necessárias, inserir a responsividade e finalmente organizar o projeto, documentá-lo e finalizá-lo.

O CSS do projeto foi feito dividindo os estilos em módulos para mais organização, do mesmo modo que o JavaScript foi também dividido em diversos arquivos para a mesma finalidade.

Os logotipos utilizados pertencem às respectivas tecnologias e são usados apenas para fins de identificação.
