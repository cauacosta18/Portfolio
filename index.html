Meu nome é Cauã Luiz Costa. Tive meu primeiro contato com a programação em 2024, quando me matriculei em um curso de informática onde aprendi sobre lógica de programação, bem como HTML e CSS.

PROJETOS

MINES

Uma recriação de um jogo de cassino online, Mines, para uma feira de matemática, nele os usuários podem escolher seu nome de perfil e jogar com um saldo de R$1000,00 fictícios, eles podem escolher a quantidade que deseja apostar e quantas bombas ter no campo, quanto mais bombas colocarem e mais quadrados revelarem, maior será o multiplicador do prêmio.

Github

PARADIGMAS DE PROGRAMAÇÃO (JAVA)

Github

CARDÁPIO ONLINE

Como resultado de um projeto realizado com o objetivo de integrar um site com um banco de dados relacional, este site simula um menu de um restaurante onde os funcionários podem cadastrar novos pratos, sobremesas, bebidas, fornecedores e outros funcionários.

Github

PROXWORK

O trabalho final do curso Jovem Programador, idealizado e criado durante o mesmo. Trata-se de um MVP de um sistema onde prestadores de serviço podem se cadastrar e ser encontrados por pessoas os procuram, direcionado para trabalhadores mais modestos. Esse MVP conta com a possibilidade de prestadores e clientes criarem suas contas, autenticação de login, pesquisa e filtragem de prestadores.

Github

COLEÇÃO DE FERRAMENTAS

Um projeto simples criado com o objetivo de praticar as linguagens HTML e CSS, este site é uma vitrine de diversas ferramentas que acho interessantes.

Github

QUIZ

Este projeto foi criado para um trabalho escolar de inglês como um extra, nele eu pratiquei as linguagens HTML, CSS e PHP. O projeto é um site onde é possível criar quizzes simples de múltipla escolha e realizá-los.
